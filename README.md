tech.bedrockstreaming.com
=========================

Bedrock Tech Blog

#### Installation

```shell
git clone https://github.com/BedrockStreaming/tech.bedrockstreaming.com.git
cd tech.bedrockstreaming.com
sudo gem install jekyll bundler
bundle install
bundle exec jekyll serve
```
